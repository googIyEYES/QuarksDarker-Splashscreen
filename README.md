
# QuarksDarker-Splashscreen
This Is A Custom Splashscreen for KDE Plasma made by [**yanekyuk**](https://store.kde.org/u/yanekyuk) on KDE Store. This REPO is used by me in a automated customization script.

<h5>Clone This Repo And `cd` into it.</h5>

```
git clone https://github.com/googIyEYES/ArchSpace-Splashscreen.git && cd ArchSpace-Splashscreen
```

<h5>Install</h5>

```
kpackagetool6 -t Plasma/LookAndFeel -i archspace.tar.gz
```

<h5>Remove</h5>

```
kpackagetool6 -t Plasma/LookAndFeel -r archspace
```
